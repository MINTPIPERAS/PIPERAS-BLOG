# Vue 3 + Vite

这是一个「前后端分离的博客系统」
前端：Vue 3 + Vue Router + Axios
后端：Node.js + Express + MongoDB + Mongoose
通信方式：REST API(JSON/ multipart-form-data)
文件存储：本地uploads/目录(封面图片）
前端只负责页面和交互，后端只负责数据、文件、业务逻辑
两者通过HTTP接口连接

简单好上手 适合新人搭建
